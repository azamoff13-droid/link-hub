# Link Hub - Antigravitasiya Uslubida 🚀

Bu loyiha Koreys tili o'qituvchisi uchun maxsus yaratilgan interaktiv "Linktree" alternativasi.

## ✨ Xususiyatlari:
- **Google Antigravity Effect**: Linklar ekranda erkin suzib yuradi.
- **Interaktivlik**: Sichqonchani link ustiga olib kelsangiz, u to'xtaydi va kattalashadi.
- **Glassmorphism**: Zamonaviy, shaffof va premium dizayn.
- **Localhost**: Loyiha hozirda mahalliy serverda ishlayapti.

## 📁 Tarkibi:
- `index.html` - Struktur va asosiy qism.
- `style.css` - Dizayn va vizual effektlar.
- `script.js` - Fizika va harakat algoritmi.

## 🚀 Ishga tushirish:
Brauzeringizda quyidagi manzilni oching:
**http://localhost:8080**

## 💡 O'zgartirish kiritish:
Agar o'z rasmingizni qo'ymoqchi bo'lsangiz, `index.html` faylidagi `<img>` tegini o'zgartiring yoki loyiha papkasiga rasm joylab nomini to'g'irlang.
